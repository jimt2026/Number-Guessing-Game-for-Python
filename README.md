# Number-Guessing-Game-for-Python
# Version 1.0
# 24/04/2026

You must guess a number from 1 to 100. 
You get 6 guesses. 
You will be told if the number you guess is higher or lower than the correct number. 
Once you guess the correct number or fail, press Enter to restart. 

Pygame is not required for this game, but Python is. 
When you open the file, it should open a terminal window which is where the game is played. If it does not, try a different Python version.
